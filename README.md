# Fullstack Application (React + FastAPI + PostgreSQL)

Полноценное fullstack-приложение с:
- **Frontend**: React + Vite
- **Backend**: FastAPI + SQLAlchemy (async)
- **Database**: PostgreSQL
- **Контейнеризация**: Docker + Docker Compose

---

# Запуск проекта
docker compose up --build

# Перезапуск
docker compose down
docker compose up --build
